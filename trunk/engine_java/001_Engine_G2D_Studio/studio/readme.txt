= g2d studio 组件介绍 =

基于 CellGameEditor 的资源管理器

该组件提供对大量的 精灵,地图,场景 进行管理的能力, 
可以在不同的cell 编辑器间共享资源，也可以用于界面设计, 地图编辑.




= xls结构 =

1.	表信息定义
	第一行为数据信息定义，该行可以根据需要写自由数据。不对数据格式要求。

2.	表数据头定义
	第二行为数据字段定义，从B列开始，每列表示一个数据字段，字段参考sql数据结构。
	
	
3.	表数据定义
	第三行到第N行为数据定义，填写数据头字段定义的数据。填写什么类型的数据参考sql数据结构。
